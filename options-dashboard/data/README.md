# Instrument list (MCX only)

The dashboard resolves **MCX** front-month futures (e.g. Silver, Crude) using Dhan’s master instrument file.

1. Download **`dhan-instrument-list_full.csv`** from Dhan’s official instrument / API documentation sources.
2. Save it here as:

   `options-dashboard/data/dhan-instrument-list_full.csv`

NSE index underlyings (NIFTY, BANK NIFTY, FIN NIFTY) **do not** need this file.
